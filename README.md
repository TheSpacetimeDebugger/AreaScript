## 🌟 عن المشروع (About)
**AreaScript** هي لغة برمجة نصية عربية تهدف إلى كسر حاجز اللغة في عالم التقنية. صُممت لتكون جسراً لكل مبدع عربي يطمح لتعلم البرمجة بلغه الأم، مع الحفاظ على قوة المنطق البرمجي الحديث (OOP) وسهولة القراءة.
> "البرمجة ليست مجرد كود، بل هي منطق ناطق بلساننا." — *إبراهيم (المطور)*
> 
## ✨ المميزات الرئيسية (Features)
 * **لغة الضاد:** البرمجة بالعربي الكامل (الأوامر، المكتبات، المتغيرات).
 * **منطق بايثون:** تعتمد على الإزاحة (Indentation) لتبسيط هيكلة الكود.
 * **برمجة الكائنات (OOP):** دعم كامل للكلاسات (كلاس)، ودوال البناء (جديد) والمرجع الذاتي (هذا).
 * **مكتبات مدمجة:** مكاتب قوية للرياضيات (جذر، أس...)، الوقت (وقت_الآن)، والنظام.
 * **محرر ويب متكامل:** بيئة تطوير متكاملة (IDE) تعمل مباشرة في المتصفح.
 * **صُنعت من الجوال:** قصة نجاح تثبت أن العزيمة أقوى من الإمكانيات.
## 🚀 البداية السريعة (Quick Start)
لا تحتاج لتثبيت أي شيء! فقط افتح ملف AreaScript.html في متصفحك وابدأ بكتابة كودك الأول:
```areascript
# مثال: نظام ترحيب ذكي
دالة ترحيب(اسم):
    اطبع("أهلاً بك يا " + اسم + " في عالم AreaScript!")

المستخدم = ادخل("أدخل اسمك: ")
ترحيب(المستخدم)

اذا طول(المستخدم) > 5:
    اطبع("ما شاء الله، اسمك له هيبة!")
والا:
    اطبع("اسمك مختصر ومميز!")

```

## 📐 مكتبة الرياضيات (Math Library)
اللغة تدعم العمليات الحسابية المتقدمة:
 * جذر(س): حساب الجذر التربيعي.
 * أس(س, ن): رفع الرقم للقوة.
 * PI: الثابت الرياضي (3.14159).
 * جيب(س)، جتا(س)، ظا(س).
